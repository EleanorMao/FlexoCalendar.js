#FlexoCalendar.js
===

###概述
- 依赖jQuery
- 可展示日历、周历、月历
- 同时提供方法输入日期即可得知该日是该月的第几周以及每月的天数

===
###使用
在html中引入jQuery.js,FlexoCalendar.js以及FlexoCalendar.css，例如

```html

    <link rel="stylesheet" href="FlexoCalendar.css">
    <script src="jq.js"></script>
    <script src="FlexoCalendar.js"></script>

```

###调用方法

```JavaScript
    <script>
      $(function(){
        $(dom).flexoCalendar();
      })
    </script>
```
